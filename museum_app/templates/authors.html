Сеть авторов

С помощью информации из Викиданных можно визуализировать связи между авторами работ.

Тип связи Викиданные Цвет Пример
Учитель-ученик Student, Student of, Educated at серый Nikolai Rimsky-Korsakov -> Alexander Glazunov
Родители Mother, Father синий Alexandre Benois -> Nikolaï Alexandrovitch Benois
Партнер Spouse, Unmarried partner розовый Leo Tolstoy + Sophia Tolstaya
Братья / сестры Sibling пурпурный Yelena Polenova + Vasily Polenov
Родственник Relative зеленый Erasmus Darwin + Charles Darwin
Крестные родители Godparent черный Friedrich Engels -> Karl Liebknecht

На графе показаны связи между людьми (самые крупные компоненты графа). Более темным цветом выделены наиболее влиятельные по конфигурации связей вершины (топ-50 VoteRank алгоритм). Цветом обозначена гендерная принадлежность.

Группа Цвет
Женщины красный
Мужчины зеленый
Иное серый

К сожалению, сеть гендерно несбалансированная, есть сильный перекос в сторону мужчин. Среди влиятельных вершин только 1 женщина, остальные часто связаны социальными связями, а не связями учитель-ученик.

Граф довольно сложный для показа, для загрузки требуется время
